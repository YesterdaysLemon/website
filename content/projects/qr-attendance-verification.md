This project extended the earlier mobile application architecture into a second attendance-focused use case.

The goal was to verify student attendance through a QR-based mobile workflow while reusing the parts of the existing stack that were already working well.

The project included:

- QR code generation and validation
- device permission handling for mobile scanning flows
- Firebase-backed data handling
- adapting an existing architecture instead of rebuilding from scratch
