Drive Safe Doha was built as a proof of concept for the Qatar Ministry of Education. The core idea was simple: make it harder for unauthorized users to access a vehicle by routing key checks through a mobile application.

The project focused on building a usable mobile flow first and then backing it with cloud data management and photo-based verification.

What mattered most in this project:

- keeping the application flow understandable for nontechnical users
- making the verification path reliable enough for a proof-of-concept demo
- validating that the architecture could be adapted into follow-on projects
