CentraID is a capstone project aimed at building an attendance solution for universities through a hybrid RFID and mobile-device system.

My focus on the project has been split between product thinking and delivery infrastructure:

- helping shape the microservice architecture
- contributing to stack decisions and feature direction
- working on the CI/CD pipeline
- containerizing backend services with Docker

It is the project that most clearly ties together software engineering and systems work.
